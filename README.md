# 🛒 쇼핑 리스트 앱

단일 HTML 파일로 동작하는 쇼핑 리스트 웹 앱입니다.

## 기능

- 항목 추가 (입력 후 `+` 버튼 또는 `Enter` 키)
- 항목 체크/언체크 (구매 완료 표시)
- 항목 삭제
- 완료 항목 일괄 삭제
- 브라우저 새로고침 후에도 데이터 유지 (localStorage 사용)

## 사용 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라)
- localStorage (데이터 영속성)
